# メモクラNeo

### みんなで協力して""さいきょう""のメモクラをつくろう

> アイデアあったら書いてね

- 議事録からタスクの抽出　カレンダーの登録
  - 認識できたタスクの一覧を出してクリックしたら登録かな
  - GoogleカレンダーAPI連携
- 参加者の認識　（だれが喋ったかの把握）
  - whisperで話者分離できるか
  - 話者分離に何かしら別のモデルをかませることも検討
-　議事録の文をクリックしたら該当の時間から動画再生
-　PDFのフォーマット化して出力可能
  - いろんなフォーマット選べてもいいかも
- 精度向上
  - 無音区間をカットできれば書き起こしがバグりづらいのは確認済
